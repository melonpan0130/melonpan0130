### 근육계통*Muscular system*

* **뼈대근육*Skeletal m.*** 뼈와 뼈를 연결하여 운동을 일으키는 근육
  * 의지를 가지고 움직일 수 있는 근육
  * 이는 곳과 닿는 곳 : 근육이 뼈와 고정되는 부분이 **이는곳*Origin***, 움직이는 쪽이 **닿는곳*Insertion***
  * 근육 이름 짓기 : 기능, 생김새, 위치, 근육섬유 배열 등에 따라 이름이 지어짐
* **심장근육*Cardiac m.*** 심장을 이루고 있는 근육, 저절로 움직임
* **민무늬근육*Smooth m.*** 주로 장기의 벽을 이루고 있는 근육. 저절로 움직이는 근육



#### 근육의 분류

근육은 조직학적 형태와 위치에 따라 가로무늬근육(뼈대근육), 심장근육, 민무늬근육으로 분류한다.

|                      | 뼈대근육<br />*Skeletal m.* | 심장근육<br />*Cardic m.*           | 민무늬근육<br />*Smooth m.* |
| -------------------- | --------------------------- | ----------------------------------- | --------------------------- |
| 가로무늬(Striations) | O                           | O                                   | X                           |
| 핵의 위치            | 세포의 가쪽                 | 세포의 가운데                       | 세포의 가운데               |
| 핵의 개수            | 다핵                        | 1~2개                               | 단핵                        |
|                      |                             | * 사이원반<br />*Intercalated disc* |                             |
